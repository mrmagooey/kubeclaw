=== Next Step: Add a Channel ===

Open the admin shell to connect a messaging channel.

Exec (interactive terminal inside the pod):

  kubectl exec -it deployment/kubeclaw-orchestrator -n {{ .Values.namespace }} -- node dist/admin-shell.js

HTTP browser UI — set these env vars in the orchestrator Deployment and expose via Service + Ingress:

  ADMIN_HTTP_PORT=9090
  ADMIN_HTTP_USERNAME=admin
  ADMIN_HTTP_PASSWORD=<your-password>

Tell it in plain English: "set up Telegram", "add Discord", etc.
It will ask for credentials, create the channel pod, and register your first group.
